## Getting Started

```bash
git clone https://github.com/didemkaraaslan/covid19watcher_youtube.git
cd covid19watcher/
npm i
npm start
# or
yarn run
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
